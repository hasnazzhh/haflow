Latihan Interaktif Flowgorithm

Soal 1: Deklarasi Variable

Tuliskan perintah untuk mendeklarasikan tiga variable: panjang, lebar, dan tinggi bertipe Integer.

Soal 2: Assignment

Tuliskan perintah untuk menghitung volume balok dari variable panjang, lebar, dan tinggi, lalu simpan hasilnya di variable volume.

Soal 3: Input Statement

Tuliskan perintah input untuk meminta pengguna memasukkan nilai panjang.

Soal 4: Output Statement

Tuliskan perintah output untuk menampilkan teks Volume balok adalah: diikuti nilai variable volume.

Soal 5: Struktur IF-ELSE

Tuliskan perintah struktur IF untuk menampilkan "Lulus" jika nilai >= 75, dan "Remedial" jika tidak.

Konsep Penting Flowgorithm

1. Deklarasi Variable: Mendefinisikan nama variable dan tipe datanya sebelum digunakan.
2. Assignment: Memberi nilai pada variable, bisa berupa operasi hitung.
3. Input dan Output: Input untuk mengambil data dari pengguna, Output untuk menampilkan hasil.
4. Struktur Kondisional (IF-ELSE): Pengambilan keputusan berdasarkan kondisi benar/salah.
5. Visualisasi Flowchart: Representasi diagram alur program untuk memudahkan pemahaman.
