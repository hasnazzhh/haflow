Panduan Lengkap Penggunaan Flowgorithm

Belajar membuat flowchart dan algoritma dengan mudah dan lengkap

1. Apa Itu Flowgorithm?

Flowgorithm adalah aplikasi yang membantu kamu membuat flowchart secara visual, sehingga memudahkan memahami cara kerja algoritma dan pemrograman. Flowchart adalah diagram yang menggambarkan alur proses secara urut dan jelas.

2. Membuat Variable (Deklarasi Variable)

Variable adalah tempat penyimpanan data sementara di dalam program. Kamu harus mendeklarasikan variable sebelum menggunakannya di Flowgorithm.

  • Buka menu Edit > Variables
  • Klik tombol Add untuk menambahkan variable baru
  • Isi nama variable, tipe data, dan nilai awal jika perlu
Contoh deklarasi variable:
  Nama: panjang
  Tipe: Integer
  Nilai awal: (kosong atau 0)

3. Mendeklarasikan Banyak Variable Sekaligus

Di Flowgorithm, kamu bisa mendeklarasikan beberapa variable sekaligus dengan memisahkan nama variable menggunakan koma.

Declare panjang, lebar, tinggi As Integer

Artinya ketiga variable tersebut bertipe Integer.

4. Aturan Penamaan Variable

  • Nama variable harus dimulai dengan huruf (a-z atau A-Z)
  • Bisa mengandung angka dan underscore (_) tapi tidak boleh diawali angka
  • Tidak boleh ada spasi di nama variable
  • Case sensitive, artinya panjang dan Panjang adalah variable berbeda
Contoh nama variable yang benar:
  nilai1
  panjang_balok
  Volume

5. Memberi Nilai pada Variable (Assignment)

Kamu bisa memberikan nilai atau mengubah nilai variable dengan blok Assignment.

Contoh assignment:
  volume = panjang * lebar * tinggi

6. Input dan Output Statement

Input (Memasukkan Data)

Input digunakan untuk menerima data dari pengguna dan menyimpannya ke variable.

Input panjang
Input lebar
Input tinggi

Output (Menampilkan Data)

Output digunakan untuk menampilkan informasi atau hasil perhitungan ke layar.

Output "Volume balok adalah: " + volume

7. Seleksi (If-Else)

Seleksi atau percabangan digunakan untuk menjalankan perintah berbeda berdasarkan kondisi.

IF nilai >= 75 THEN
    Output "Lulus"
ELSE
    Output "Remedial"
ENDIF

8. Contoh Lengkap: Flowchart Menghitung Volume Balok

Langkah-langkah:

  1. Declare variable: panjang, lebar, tinggi, volume (semua Integer)
  2. Input nilai panjang, lebar, dan tinggi dari pengguna
  3. Hitung volume dengan rumus: volume = panjang × lebar × tinggi
  4. Output hasil volume ke layar
Declare panjang, lebar, tinggi, volume As Integer

Input panjang
Input lebar
Input tinggi

volume = panjang * lebar * tinggi

Output "Volume balok adalah: " + volume

9. Contoh Kode Flowgorithm (XML) untuk Import

<?xml version="1.0" encoding="UTF-8"?>
<flowgorithm fileversion="2.19">
  <attributes>
    <attribute name="name" value="Hitung Volume Balok"/>
    <attribute name="variables">
      <variable name="panjang" type="Integer"/>
      <variable name="lebar" type="Integer"/>
      <variable name="tinggi" type="Integer"/>
      <variable name="volume" type="Integer"/>
    </attribute>
  </attributes>
  <main>
    <declare name="panjang" type="Integer"/>
    <declare name="lebar" type="Integer"/>
    <declare name="tinggi" type="Integer"/>
    <declare name="volume" type="Integer"/>
    
    <input variable="panjang" prompt="Masukkan panjang:"/>
    <input variable="lebar" prompt="Masukkan lebar:"/>
    <input variable="tinggi" prompt="Masukkan tinggi:"/>
    
    <assignment variable="volume" expression="panjang * lebar * tinggi"/>
    
    <output expression=""Volume balok adalah: " + volume"/>
  </main>
</flowgorithm>
